◆ CanPlaceShip()

bool CanPlaceShip ( GameBoard * b,
int r,
int c,
int size,
bool vertical )

Vérifie si un navire peut être placé à une position donnée.

  • Contrôle les limites de la grille et s'assure qu'aucun autre navire n'est présent dans les cellules adjacentes (incluant les diagonales).
  • Paramètres
    bLe plateau de jeu.
    r,cCoordonnées de départ (ligne, colonne).
    sizeTaille du navire.
    verticalOrientation.
    Renvoie
    true si le placement est valide.

◆ ClearBoard()

void ClearBoard ( GameBoard * b)

Réinitialise complètement le plateau de jeu.

  • Cette fonction parcourt l'intégralité de la grille pour vider chaque cellule et remet le compteur de navires restants à zéro.
  • Paramètres
    bUn pointeur vers la structure GameBoard à réinitialiser.

◆ ProcessCPUTurn()

void ProcessCPUTurn ( GameBoard * player,
bool * playerTurn,
CPUState * cpu,
AIDifficulty diff )

Gère le tour de l'IA selon la difficulté choisie.

  • Détermine la cible (aléatoire, pile de ciblage ou carte de probabilité), traite le résultat du tir (touche, coulé ou manqué) et met à jour l'état du jeu ainsi que le tour des joueurs.
  • Paramètres
    playerplateau du joueur.
    playerTurnpointeur vers le booléen contrôlant le tour du joueur.
    cpuétat interne de l'IA.
    diffniveau de difficulté (FACILE, NORMAL, EXPERT).
